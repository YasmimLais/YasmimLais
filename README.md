Estudante de **Engenharia da Computação** na UFPE e **Análise e Desenvolvimento de Sistemas**.

Desenvolvo minha trajetória na intersecção entre o físico e o digital. Através da graduação, busco unir uma visão sistêmica onde o hardware fornece a base, o software entrega a inteligência e a engenharia garante a viabilidade de soluções completas e escaláveis.

Hardware  →  Software  →  Dados
